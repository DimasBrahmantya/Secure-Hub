a. Nama Aplikasi : Secure Hub

b. Deskripsi Aplikasi :
SecureHub merupakan sebuah sistem berbasis web yang dirancang untuk membantu administrator dalam melakukan pemantauan, analisis, dan pengelolaan keamanan terhadap aktivitas pengguna maupun URL yang berpotensi mengandung ancaman phishing. Sistem ini menyediakan berbagai fitur keamanan terintegrasi, seperti analisis URL, monitoring aktivitas, manajemen pengguna, serta mekanisme backup dan restore data. Dengan tampilan antarmuka yang konsisten dan mudah dipahami, SecureHub bertujuan untuk memberikan lingkungan operasional yang aman, terstruktur, dan efisien.
